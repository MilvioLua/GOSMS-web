O que significa cada sinalizador de status de entrega?

ENTREGUE Conforme informado pela operadora de rede do destinatário, a mensagem foi entregue ao aparelho. ACEITO A mensagem foi enviada com sucesso para a operadora, mas ainda não foi entregue ao aparelho. Se você verificar o log de sms, poderá obter mais informações sobre a situação específica de cada destinatário (por exemplo, 'Assinante Ausente'). EXPIRED A mensagem demorou muito para que a linha do destinatário estivesse disponível para entrega, portanto expirou REJEITADO O número é inválido ou não existe em nenhuma rede. (NB: Esta é a ÚNICA situação em que o crédito é estornado pelo sistema). UNDELIVERABLE Pode ocorrer devido a vários motivos, tais como: - A memória da caixa de entrada do destinatário está cheia. - Em roaming e o operador não prevê reencaminhamento de sms. - A linha está atualmente desativada. - O destinatário tem algum tipo de configuração restritiva/de restrição. NOTA: Para alguns SMS com status REJEITADO, os créditos de SMS gastos são automaticamente revertidos para o seu saldo pelo sistema. Mas para todos os outros casos, os créditos gastos são finais..
